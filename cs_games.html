Use these games to reinforce what you’re teaching in Intro to CS or AP CSP: decision-making, boolean logic, and vocabulary.

Core Games

Concept-focused activities for learning and practice.

// If / Else Adventure

Conditionals

Choose a branching story powered by if, else if, and else logic. Your path changes the outcome and your score.

Mode: Single-player • Est. 5–10 minutes

// Boolean Battle

Logic

Solve quick-fire boolean logic puzzles and practice truth tables.

Mode: Single-player • Est. 5 minutes

// Binary Blaster

Conversions

Convert between binary and decimal with two modes: Binary → Decimal and Decimal → Binary.

Mode: Single-player • Est. 5 minutes

// Algorithm Assembly

Sorting

Arrange shuffled steps to build Bubble Sort, Selection Sort, and Insertion Sort correctly.

Mode: Single-player • Est. 6 minutes

// Debugging Dash

Debugging

Race the timer to spot the bugs hiding in short code snippets.

Mode: Single-player • Est. 5 minutes

// Variable Vault

Data Types

Choose the best variable name, type, and value for each scenario.

Mode: Single-player • Est. 6 minutes
