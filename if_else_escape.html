Story Rules

  1. Enter your name and pick a Level.
  2. Levels are tied to course rigor:
    • Level 1: Intro to Computer Science
    • Level 2: AP Computer Science Principles
    • Level 3: AP Computer Science A
    • Level 4: Honors Data Structures & Algorithms
    • Level 5: College-level Computer Science
  3. Higher levels introduce more complex decisions and a bigger point swing. You can earn or lose points based on the path you take.
  4. Read the scenario and the if / else code. The code block shows the current variable values that drive the story.
  5. Choose which path to take. Each branch has a different outcome.
  6. You have 3 lives. Risky choices can cost 1 life.
  7. Each story has checkpoints (scoring a goal, finding treasure, resolving a conflict, etc.). If you lose all lives, you can restart from your last checkpoint with full lives, but your score beyond that checkpoint resets.
  8. Each level is its own story arc with branching choices (up to three options per step).
